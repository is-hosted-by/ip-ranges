Networks Microdata Schema

Microdata schema for describing a network and it's IP ranges.

Property Expected Type Description
name Text The name of the network.
sub-networks Network A network can be composed of many sub-networks.
ranges Address Range A network has many address ranges identifying addresses that belong to it.
source URL The network location of the resource used to source the IP range information in this network.

Address Range

Microdata schema for describing a sub-network of IP ranges.

Property Expected Type Description
id Text Optional unique identifier of the sub-network.
cidr Text An optional CIDR notation representing an IP range. You MUST specify only one cidr or from property. The cidr text MUST be a wellformed IPv4 or IPv6 CIDR notation.
from Text An optional IP number representing the lower bound of IP addresses contained in this IP range. You MUST specify only one cidr or from property. If you specify the from property you MUST also specify a to property. The from text MUST be a wellformed IPv4 or IPv6 address.
to Text An optional IP number representing the upper bound of IP addresses contained in this sub-network. If you specify the to property you MUST also specify a from property. The to text MUST be a wellformed IPv4 or IPv6 address.
